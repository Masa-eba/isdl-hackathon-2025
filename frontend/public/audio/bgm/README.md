# BGMファイル配置ディレクトリ

以下のBGMファイルを配置してください：

## 必要なBGMファイル
- title.mp3 - タイトル画面用
- prologue.mp3 - プロローグ用  
- main_game.mp3 - メインゲーム用
- conversation.mp3 - 会話シーン用
- epilogue.mp3 - mainzmaiエピローグ用
- ending.mp3 - エンディング用
- menu.mp3 - メニュー用

## 推奨仕様
- フォーマット: MP3
- ビットレート: 128kbps以上
- 長さ: 2-3分程度（ループ再生）
- 音量: 統一されたレベル

## ファイル配置例
```
public/audio/bgm/
├── title.mp3
├── prologue.mp3
├── main_game.mp3
├── conversation.mp3
├── epilogue.mp3
├── ending.mp3
└── menu.mp3
```
